## Midterm Lab-Task 3 - Data Prep and Cleaning and Modeling w Power Query
- It means using Power Query in Excel/Power BI to prepare raw data for analysis:

## step by step

# step 1 Import Data
Load all sheets (Students, Courses, Programs, etc.) into Power Query.
# step 2 Clean the Data
Remove duplicates and blank rows.
Fix inconsistent formats (e.g., names, emails, grades).
Replace "INC" or missing grades with null or default values.
# step 3 Transform Columns
Split full names if needed.
Change data types (e.g., time to time format, year level to number).
Create calculated columns (e.g., duration = TimeEnd − TimeStart).
# step 4 Merge & Relate Tables
Use keys like ProgramID, CourseID, StudentID to merge tables.
Build relationships for modeling (Student ↔ Program ↔ Course).
# step 5 Load to Excel or Power BI
Load cleaned data for pivot tables, charts, or dashboards.
